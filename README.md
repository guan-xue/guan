# 管学

## 什么是管学？

管学是一门后现代主义学问。主要讨论了“管与不管”相关的问题。“早该管管了”是其信徒常用的口号。“管天管地管空气”是管学的终极目标，也就是如何抽象出一门统一的理论来管一切事物。

## 管学的起源

早2000多天前，奥斯卡终身成就奖获得者，成龙，在博鳌论坛上就曾直言："中国人是要管"。其被后世学者公认为是管学的祖师爷。
